Medical application for the eyes

This application is a demo version and is aimed at improving the diagnostic processes of ophthalmic analyzes. This application will support preliminary diagnostics on angular velocity eye pupils measuring.

Algorithm & processing

Deselection of photos from series can sightly reduce eye-tracking algorithm accuracy. Perform this action only if you sure algorithm has failed to track an eye pupil correctly for one or more photos.

Photo processing and process costs

Before starting the process of finding the coordinates of pupils, the taken or uploaded photos should be reduced to the minimum resolution in order to increase the speed of processing them by the OpenCV library

Step2

Step3

Take a photoshoot in a well-lit environment, make sure there are no glares on the pupils and the screen of the phone.
